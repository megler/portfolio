Zillow Scraper

A Python web scraper focused on extracting Zillow information and adding to a Google Form for processing.

A quick overview

Given a set of parameters, scrape Zillow and fill out Google Form with gathered data. This project uses Beautiful Soup, Selenium and Google Forms.

Project Links

This is what Zillow Scraper will do

Features

Given a Zillow search URL containing desired search params (eg. price, location, etc), scrape all returned results and add them to a Google form.

This allows a user to quickly gather a lot of information on a given area and import into the program of choice for data analysis. This project could be easily modified to import scraped data into a database or spreadsheet vs a Google Form.

This Tool Uses:
  • Beautiful Soup
  • Selenium
  • Google Forms
