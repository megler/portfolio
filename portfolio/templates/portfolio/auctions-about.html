Features
  • User Authentication
  • Custom Admin
  • Relational SQL Database
  • User specific watchlists
  • Contextual error messages
  • Custom gravatars for logged in users
  • Comments section where logged in users can comment on auction items
  • Logged in users can add, bid and if criteria are met, end an auction
  • The homepage lists all active auctions. Once an auction ends, it will no longer show up on this page.
  • Categories for various types of items (eg. Brooms, Wands, etc)
  • An individual listing page will display all relevant information, allow a logged in user to place a bid, add a comment and also add to their watchlist.
  • If a logged in user is on the listing page of an auction now ended and they are the highest bidder, they will be notified. Otherwise they will have an alert the auction has ended.
  • If item is already in user's watchlist, an alert will says "Item currently in watchlist" and give user the option to click a link to see their watchlist.
  • In the watchlist, a user can delete anything from the list.
