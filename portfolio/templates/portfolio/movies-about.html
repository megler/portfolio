This is what Movie Tracker will do

Features
  • Perform all CRUD (Create, Read, Update and Delete) functions
  • Backend SQL database
  • Movie posters and description are pulled from The Movie Database API
  • Add or Update both your personal rating of the movie as well as its review.
  • Movies are organized from least to move favorite
