Stock and News Alerts

A Python application utilizing external API's to send text and email alerts for stock prices and news.

A quick overview

Get a text or email when your chosen stock(s) move more than N% or have relevant news.

Project Links
  • Check out the code: Github

This is what Stock News Alerts will do

Features

Given a set of parameters, use Twilio, NewsAPI and AlphaAdvantageAPI to send a text with the top 3 headlines if a chosen stock is up or down over a given percentage (5%).

This allows a user to be notified via text and/or email if their stock makes a move or has news. This allows you to stay on top of your stock portfolio without a lot of fuss.

This Tool Uses:
  • Twilio
  • NewsAPI
  • Alpha Vantage API
